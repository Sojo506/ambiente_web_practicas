Ejercicio 3: Estructura de Control de Flujo (Condicionales)

Diseña una página web que solicite al usuario ingresar su edad.

Mediante una estructura condicional, evalúa la edad:

Si el usuario es mayor de 18, muestra el mensaje: "Eres mayor de edad."

En caso contrario (si es 18 o menor), muestra el mensaje: "Eres menor de edad."
