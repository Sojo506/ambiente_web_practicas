Ejercicio 4: Manipulación de Arreglos (Arrays)

Crea un arreglo (array) de objetos que almacene información de varios estudiantes, incluyendo las propiedades nombre, apellido y nota.

Recorre el arreglo usando un ciclo for o forEach e imprime los nombres y apellidos de cada estudiante dentro de un elemento div en la página.

Calcula el promedio de las notas de todos los estudiantes y muestra el resultado justo al finalizar la lista.
